Электронный учебник создается по общеобразовательной программе "Стандарт. 11 класс" для учеников, начинающих изучать информатику с 9 класса.

11.1 1 СЕНТ-31 ОКТ
===================

Системы обработки табличных данных. Табличный процессор.
--------------------------------------------------------------

.. raw:: html

    <div class="admonition note">
        <p class="first admonition-title">Цели курса:</p>
        <p class="last"><b>Научиться</b>       
            <b>объяснять</b>: <br />
            <ul style="text-align: left;">
            <li>правила записи абсолютных, относительных и смешанных ссылок на ячейки и диапазоны ячеек; </li>
            <li>правила записи формул и использования адресов ячеек и диапазонов в формулах; </li>
            <li>правила преобразования абсолютных, относительных и смешанных ссылок при копировании формул; </li>
            </ul>
            <b>использовать</b>: <br />
            <ul style="text-align: left;">
            <li>мастер диаграмм; </li>
            <li>Автофильтры для выбора данных из таблиц по простым критериям; </li>
            <li>Сводные таблицы для комплексного анализа табличных данных; </li>
            </ul>
            <b>описывать</b>: <br />
            <ul style="text-align: left;">
            <li>понятие электронной книги, листа, строки, столбца, ячейки, диапазона ячеек; </li>
            <li>способы навигации листом и книгой; </li>
            <li>форматы данных: числовой, денежный, текстовый, формат даты; </li>
            <li>ввод данных различных форматов и управления форматом ячеек; </li>
            <li>виды ошибок при вводе данных и формул и способы их устранения; </li>
            <li>назначение основных панелей инструментов табличного процессора и кнопок на них; </li>
            <li>критерии определения типа диаграммы, которая будет оптимально отображать один или несколько рядов данных; </li>
            <li>цель и способы фильтрации данных в таблицах; </li>
            <li>назначение сводных таблиц и средств нахождения промежуточных итогов; </li>
            <li>назначение условного форматирования данных; </li>
            </ul>
            <b>уметь</b>: <br />
            <ul style="text-align: left;">
            <li>перемещаться по листу и книге; </li>
            <li>вводить данные и формулы в ячейки и редактировать их содержимое; </li>
            <li>выделять диапазоны ячеек с заданным адресом; </li>
            <li>форматировать данные, ячейки и диапазоны ячеек; </li>
            <li>копировать, перемещать и удалять содержимое ячеек и диапазонов ячеек; </li>
            <li>записывать абсолютные, относительные и смешанные ссылки на ячейки и диапазоны ячеек; </li>
            <li>определять тип диаграммы, которая лучше всего отобразит один или несколько рядов данных; </li>
            <li>изменять тип диаграммы; </li>
            <li>задавать диапазон входных данных для диаграммы и диапазон данных для каждого ряда; </li>
            <li>настраивать параметры отображения диаграммы, области данных и рядов данных; </li>
            <li>сортировать данные в таблицах по значениям одного или нескольких полей; </li>
            <li>выполнять условное форматирование данных; </li>
            <li>применять основные математические, статистические, логические функции для анализа и обработки данных; </li>
            </ul></p>
            <b>приводить примеры</b>: <br />
            <ul style="text-align: left;">
            <li>математических, статистических, логических функций табличного процессора. </li>
            </ul></p>
        </div>
